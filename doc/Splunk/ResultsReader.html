Splunk::ResultsReader

This class allows clients to enumerate events that are the results of a streaming search via the Splunk::Jobs::create_stream call. This enumeration is done in a ‘chunked’ way - that is the results are not all streamed into a large in-memory JSON data structure, but pulled as ResultsReader::each is called.
